# CanAnnulInvoice

Result of the pre-cancellation validator of an invoice: whether it can be cancelled, the blocking reasons (when it cannot) and whether the cancellation will create an additional VeriFactu record.


## Fields

| Field                                                                                                             | Type                                                                                                              | Required                                                                                                          | Description                                                                                                       | Example                                                                                                           |
| ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| `canAnnul`                                                                                                        | *bool*                                                                                                            | :heavy_check_mark:                                                                                                | Indica si la factura puede anularse en su estado actual.                                                          | false                                                                                                             |
| `reasons`                                                                                                         | array<*string*>                                                                                                   | :heavy_check_mark:                                                                                                | Blocking reasons when `can_annul` is `false`. Empty `[]` when the invoice can be cancelled.                       | [<br/>"La factura está en estado Borrador y no puede ser anulada"<br/>]                                           |
| `willCreateVerifactu`                                                                                             | *bool*                                                                                                            | :heavy_check_mark:                                                                                                | Indicates whether the cancellation will generate a cancellation record in VeriFactu.                              | false                                                                                                             |
| `info`                                                                                                            | array<*string*>                                                                                                   | :heavy_check_mark:                                                                                                | Additional informational messages about the cancellation (non-blocking warnings). Empty `[]` when there are none. | [<br/>"La factura no tiene registro VeriFactu de alta. Se anulará sin registro VeriFactu."<br/>]                  |
