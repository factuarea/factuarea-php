# Method

Preferred payment method. If `direct_debit`, the client must have at least one bank account marked as default.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `BankTransfer` | bank_transfer  |
| `DirectDebit`  | direct_debit   |
| `Cash`         | cash           |
| `CreditCard`   | credit_card    |
| `Check`        | check          |
| `Paypal`       | paypal         |
| `Other`        | other          |
