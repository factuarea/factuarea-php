# TaxTotals

Aggregated totals of a set of lines. Returned by `POST /v1/taxes/calculate-totals`. Total formula: `subtotal + total_vat + total_surcharge − total_retention`.


## Fields

| Field                                                                       | Type                                                                        | Required                                                                    | Description                                                                 | Example                                                                     |
| --------------------------------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- | --------------------------------------------------------------------------- |
| `subtotal`                                                                  | *float*                                                                     | :heavy_check_mark:                                                          | Sum of the taxable bases of all lines (EUR).                                | 250                                                                         |
| `totalVat`                                                                  | *float*                                                                     | :heavy_check_mark:                                                          | Sum of the VAT of all lines.                                                | 52.5                                                                        |
| `totalSurcharge`                                                            | *float*                                                                     | :heavy_check_mark:                                                          | Sum of the equivalence surcharge of all lines.                              | 0                                                                           |
| `totalRetention`                                                            | *float*                                                                     | :heavy_check_mark:                                                          | Sum of the withholding (IRPF) of all lines.                                 | 15                                                                          |
| `total`                                                                     | *float*                                                                     | :heavy_check_mark:                                                          | Total general.                                                              | 287.5                                                                       |
| `lines`                                                                     | array<[Components\TaxTotalsLine](../../Models/Components/TaxTotalsLine.md)> | :heavy_check_mark:                                                          | Calculated breakdown for each received line, in the same order.             |                                                                             |
