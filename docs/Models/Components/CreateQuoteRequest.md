# CreateQuoteRequest

Public REST API v1 — POST /v1/quotes.

Required body: `client_id`, `series_id`, `issued_on`, `valid_until`,
`lines[]` (min 1). Optional: `notes`, `terms`, `metadata` (≤50 keys,
≤500 chars/value — VO `Metadata`).
